# 📁 File & Directory Commands

الدرس ده بيغطي كل أوامر اللينكس الأساسية اللي بتستخدمها يوميًا في التنقل، إنشاء ملفات، تعديل، نقل، نسخ، مسح... إلخ
ركز في كل أمر لأنك هتحتاجه في أي شغل سواء Admin أو DevOps أو SecOps

---

## 🧭 التنقل جوه الفايل سيستم

```bash
pwd              # بتطبع المسار الحالي
cd /etc          # تدخل على فولدر معين
cd ~             # تدخل على الهوم
cd ..            # تطلع لفوق (الديركتوري الأب)
cd -             # ترجع للمكان اللي كنت فيه قبل كده
````

💡 لو عايز توصّل لمكان بسرعة:

```bash
cd /var/log/nginx/
```

---

## 📂 إنشاء ملفات وفولدرات

```bash
mkdir myfolder               # فولدر جديد
mkdir -p folder1/folder2     # يعمل الاتنين مرة واحدة
touch file.txt               # ملف فاضي جديد
```

---

## 📄 عرض الملفات والفولدرات

```bash
ls                        # يعرض اللي جوه
ls -l                     # عرض تفصيلي (permissions, size, date...)
ls -a                     # يعرض الملفات المخفية
ls -lh                    # human-readable size
ls -ltr                   # من الأقدم للأحدث
tree                      # عرض الشجرة (لو مش عندك نزله بـ apt install tree)
```

💡 alias مفيد:

```bash
alias ll='ls -alF'
```

---

## 📋 نسخ ونقل الملفات

```bash
cp file1.txt file2.txt        # نسخ
cp -r dir1 dir2               # نسخ فولدر بمحتواه
mv file.txt /tmp/             # نقل
mv old.txt new.txt            # إعادة تسمية
```

---

## ❌ مسح الملفات والفولدرات

```bash
rm file.txt                   # حذف ملف
rm -r myfolder/               # حذف فولدر بمحتواه
rm -rf *                      # حذف كل حاجة (خطر ⚠️)
```

🧠 مفيش سلة مهملات! لو مسحت حاجة خلاص راحت
اعمل نسخ احتياطي قبل ما تمسح حاجة مهمة

---

## 📌 قراءة محتوى الملفات

```bash
cat file.txt                  # يعرض الملف كله
less file.txt                 # يعرض بشكل page-by-page (q للخروج)
more file.txt                 # زيه تقريبًا
head -n 10 file.txt           # أول 10 سطور
tail -n 10 file.txt           # آخر 10 سطور
tail -f /var/log/syslog       # متابعة اللوج في الوقت الحقيقي
```

---

## 🔍 البحث عن الملفات

```bash
find . -name "*.log"              # كل ملفات اللوج في المسار الحالي
find /etc -type f -name "hosts"   # دور على ملف معين في /etc
locate nginx.conf                 # أسرع من find (بس لازم تعمل updatedb)
```

💡 قبل ما تستخدم `locate`:

```bash
sudo updatedb
```

---

## 🔀 إعادة توجيه وإخراج

```bash
echo "test" > file.txt         # يكتب جملة جوه ملف (يمسح القديم)
echo "line" >> file.txt        # يزود جملة جديدة في الآخر
cat file.txt | grep "ERROR"    # يفلتر سطور فيها ERROR
```

---

## 🧠 مثال عملي من الشغل:

#### سكريبت بسيط يشيك على ملفات مهمة:

```bash
#!/bin/bash

for file in /etc/hosts /etc/passwd /etc/resolv.conf
do
  if [ -f "$file" ]; then
    echo "✔️ $file موجود"
  else
    echo "❌ $file مش موجود"
  fi
done
```

اعمله:

```bash
chmod +x check_files.sh
./check_files.sh
```

## ❗ أخطاء شائعة:

* استخدام `rm -rf /` بالغلط ➜ بيبوظ السيستم
* نسيان المسار في `mv` ➜ ممكن تكتب فوق ملف تاني
* استخدام `cp` بدون `-r` مع فولدر ➜ مش هيشتغل
* نسيان `sudo` مع ملفات في `/etc`

---

## 🧪 تاسكات ليك:

* جرب تعمل فولدر فيه ملفات مختلفة
* انسخهم لفولدر تاني
* احذف ملف معين
* ابحث عن كل ملفات `.conf` في `/etc`
* اعمل ملف فيه لوج وجرب `tail -f`

## 📁 1. الأساسيات (Basic Commands)

### 🧩 الأوامر اللي لازم تحفظهم من غير ما تفكر:

### 📂 `pwd` – بيطبعلك مكانك الحالي (Current Directory)

```bash
pwd
```

✅ بتقف على السيستم ومش عارف إنت فين؟
📍 استعمل `pwd`، هيرجعلك مثلاً: `/home/eslam`

---

### 📂 `ls` – بيعرض محتوى الفولدر اللي إنت واقف فيه

```bash
ls
```

✅ لو عايز تشوف الملفات والفولدرات
💡 لو عايزهم بتفاصيل:

```bash
ls -l
```

💡 ولو عايز تشوف الحاجات المخفية:

```bash
ls -a
```

🧠 سؤال سريع:
لو عايز تعرض كل الملفات المخفية بتفاصيلها، تستخدم أمر إيه؟

---

### 🔁 `cd` – تدخل فولدر تاني

```bash
cd Documents
```

✅ بتدخل جوه فولدر
💡 لو عايز ترجع خطوة ورا:

```bash
cd ..
```

💡 لو عايز تروح للـ Home بتاعك:

```bash
cd ~
```

💡 لو كتبت بس `cd` هيرجعك تلقائي للـ home

---

### 📄 `touch` – بيعمل ملف فاضي

```bash
touch file.txt
```

✅ أنشأت ملف اسمه `file.txt`

---

### 📁 `mkdir` – بيعمل فولدر جديد

```bash
mkdir myfolder
```

💡 لو عايز تعمل أكتر من فولدر مرة واحدة:

```bash
mkdir folder1 folder2 folder3
```

---

### ❌ `rm` – يمسح ملف أو فولدر

```bash
rm file.txt
```

⚠️ خطير، بيحذف من غير تأكيد
💡 لو عايز تمسح فولدر فيه حاجات جواه:

```bash
rm -r myfolder
```

---

### 📋 `cp` – نسخ ملفات وفولدرات

```bash
cp file1.txt backup.txt
```

💡 لو عايز تنسخ فولدر كله بمحتواه:

```bash
cp -r ../linux ./
```

📌 دا بينسخ فولدر اسمه `linux` من مستوى أعلى، ويحطه في مكانك الحالي
✅ دا أمر حقيقي بنستخدمه في الشغل كل يوم

---

### 🔄 `mv` – لنقل أو إعادة تسمية

```bash
mv old.txt new.txt
```

💡 أو تنقل فايل من مكان لمكان:

```bash
mv file.txt /home/eslam/backup/
```

---

### 📖 `cat` – يعرض محتوى ملف

```bash
cat notes.txt
```

💡 لو عايز تقرأ ملفات كبيرة بـ راحة أكتر:

```bash
less notes.txt
```

### 🧠 مراجعة سريعة (جاوب على نفسك):

1. إزاي تعرض كل الملفات (بما فيهم المخفيين)؟
2. لو عايز تدخل فولدر وترجع ورا، تستخدم أمرين إيه؟
3. إزاي تنسخ فولدر كامل بمحتوياته؟
4. الفرق بين `cat` و `less`؟
5. إيه الأمر اللي تقدر بيه تسأل لينكس عن أي أمر تاني؟
جاوبك جاي بالمصري وبشكل عملي 👇

---

🔍 **إزاي تعرض كل الملفات (بما فيهم المخفيين)؟**

```bash
ls -a
```

✅ `ls` بيعرض الملفات
✅ `-a` معناها all حتى المخفيين (اللي بيبدأوا بـ .)

---

📂 **لو عايز تدخل فولدر وترجع ورا، تستخدم أمرين إيه؟**

* تدخل فولدر:

```bash
cd foldername
```

* ترجع ورا (لفولدر قبله):

```bash
cd ..
```

📝 `cd` = change directory
📝 `..` = معناه parent directory

---

📦 **إزاي تنسخ فولدر كامل بمحتوياته؟**

```bash
cp -r source_folder destination_folder
```

✅ `-r` معناها recursive يعني ينسخ كل اللي جواه حتى لو فيه فولدرات تانية
🧠 مثال:

```bash
cp -r myproject /home/eslam/backup
```

---

📄 **الفرق بين `cat` و `less`؟**

* `cat`:

  * بيطبع كل محتوى الملف مرة واحدة
  * مفيد للملفات الصغيرة
  * مالوش تحكم في التصفح

```bash
cat filename.txt
```

* `less`:

  * بيعرض الملف صفحة صفحة
  * تقدر تطلع وتنزل
  * مفيد للملفات الكبيرة

```bash
less filename.txt
```

⬆️⬇️ استخدم الأسهم للتصفح
اضغط `q` علشان تخرج

---

❓ **إيه الأمر اللي تقدر بيه تسأل لينكس عن أي أمر تاني؟**

```bash
man command
```

✅ `man` = manual
🧠 مثال:

```bash
man cp
```

هيفتح لك دليل فيه كل تفاصيل الأمر وأوبشناته

ولو عايز إجابة سريعة بدون تفاصيل طويلة:

```bash
command --help
```

مثال:

```bash
cp --help
```
